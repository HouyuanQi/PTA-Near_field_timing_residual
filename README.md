# PTA-Near_field_timing_residual
A Mathematica code to calculate the timing residual of a pulsar near a binary black hole, including non-radiative near field effect up to second order. The data of the figures in the paper is also presented.

-Timing Residual.nb: Mathematica code that defines and calculates all the functions and generates numerical data as CSV files.
-Plot.ipynb: Jupyter notebook that reads CSV files and produces pdf figures that are presented.
-Numerical_Data/: Pre-generated numerical data for all figures. Data generated by Timing Residual.nb will be saved into this folder.
-Figure/: Figures that are presented in the paper. Figure generated by Plot.ipynb will be saved into this folder.
